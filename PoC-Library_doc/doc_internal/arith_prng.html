Authors: Martin ZabelEntity: Pseudo-Random Number Generator (PRNG).

Description:

This module implementes a Pseudo-Random Number Generator (PRNG) withconfigurable bit count (BITS). This module uses an internal list of FPGAoptimized polynomials from 3 to 168 bits. The polynomials have at most 5 tappositions, so that long shift registers can be inferred instead of singleflip-flops.The generated number sequence includes the value all-zeros, but not all-ones.
