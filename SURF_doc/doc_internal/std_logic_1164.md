- resolved <font id="function_arguments">(s : std_ulogic_vector) </font> <font id="function_return">return std_ulogic </font>
**Description**
  Resolution function.
  If S is empty, returns 'Z'.
  If S has one element, return the element.
  Otherwise, 'U' is the strongest.
       then  'X'
       then  '0' and '1'
       then  'W'
       then  'H' and 'L'
       then  'Z'.
