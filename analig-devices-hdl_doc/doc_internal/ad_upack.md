 Unpacker:
   - unpack O_W number of data units from I_W number of data units
   - data unit defined in bits by UNIT_W e.g 8 is a byte
 Constraints:
   - O_W <= I_W
   - LATENCY 1 
   - no backpressure
 Data format:
  idata  [U(I_W-1) .... U(0)]
  odata  [U(O_W-1) .... U(0)]
 e.g
  I_W = 6
  O_W = 4
  UNIT_W = 8
  idata : [B5,B4,B3,B2,B1,B0],[B11,B10,B9,B8,B7,B6]
  odata :                     [B3,B2,B1,B0],[B7,B6,B5,B4],[B11,B10,B9,B8]
