SPDX-License-Identifier: Apache-2.0 AES low-bandwidth pseudo-random number generator for register clearing This module uses an LFSR connected to a PRINCE S-Box and PRESENT permutation to generate pseudo-random data for the AES module for clearing registers. The LFSR can be reseeded using an external interface.
