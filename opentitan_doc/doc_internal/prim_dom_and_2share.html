SPDX-License-Identifier: Apache-2.0 Domain-Oriented Masking GF(2) Multiplier with 2-shares ref: Higher-Order Side-Channel Protected Implementations of Keccak https://eprint.iacr.org/2017/395.pdf q0 = a0 & b0 + (a0 & b1 + z) q1 = a1 & b1 + (a1 & b0 + z) () ==> registered all input should be stable for two clocks as the output is valid after a clock For z, it can use other slice from the state as it is fairly random w.r.t the current inputs. General formula of Q in the paper Qi = t{i,i} + Sig(j>i,d)(t{i,j}+Z{i+j(j-1)/2}) + Sig(j(i-1)/2}) for d=1 (NumShare 2 for first order protection) Q0 = t{0,0} + Sig(j>0,1)(t{0,j}+Z{j(j-1)/2}) + Sig(j<0,d)(..) = a0&b0 + (a0&b1 + z0 + 0) Q1 = t{1,1} + sig(j>1,1)(…) + sig(j<1,1)(t{1,j} + Z{j}) = a1&b1 + (0 + a1&b0 + z0)
