SPDX-License-Identifier: Apache-2.0 This module can be used as a "gadget" to adapt the native 32bit width of the EDN network locally to the width needed by the consuming logic. For example, if the local consumer needs 128bit, this module would request four 32 bit words from EDN and stack them accordingly. The module also uses a req/ack synchronizer to synchronize the EDN data over to the local clock domain. Note that this assumes that the EDN data bus remains stable between subsequent requests.
