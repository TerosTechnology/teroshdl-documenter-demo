Copyright lowRISC contributors. Licensed under the Apache License, Version 2.0, see LICENSE for details. SPDX-License-Identifier: Apache-2.0 Testbench module for prim_fifo_sync. Intended to be used with a formal tool. This formal testbench instantiates a set of differently parameterized FIFOs:

  • a depth 0 pass through FIFO
  • depth 1, 7, 8, 15, 16 pass through FIFOs
  • depth 1, 7, 8, 15, 16 non-pass through FIFOsData/depth value checks are enabled up to depth 8 in order to constrain theruntime.
