SPDX-License-Identifier: Apache-2.0 RISC-V Platform-Level Interrupt Controller compliant INTC Current version doesn't support MSI interrupt but it is easy to add the feature. Create one external register and connect qe signal to the gateway module (as edge-triggered) Consider to set MAX_PRIO as small number as possible. It is main factor of area increase if edge-triggered counter isn't implemented. Verilog parameter MAX_PRIO: Maximum value of interrupt priority
