SPDX-License-Identifier: Apache-2.0 RISC-V Platform-Level Interrupt Generator for Target This module basically doing IE & IP based on priority and threshold_i. Keep in mind that increasing MAX_PRIO affects logic size a lot. The module implements a binary tree to find the maximal entry. the solution has O(N) area and O(log(N)) delay complexity, and thus scales well with many input sources.
